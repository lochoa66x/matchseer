# Phase 0 Naming Shortlist

Status:
- Superseded by `phase-0-naming-oracle-sports.md`.
- This file is kept as the first sky-themed naming round for reference only.

## Naming Goal

The name should feel:
- Modern
- Fun
- Sports-friendly
- Forecast-oriented
- Non-betting
- Expandable beyond World Cup into NFL Fantasy
- Easy to say in English, Spanish, and French

Avoid names that sound like:
- Betting products
- Sportsbooks
- Crypto apps
- Generic score trackers
- Overly serious analytics platforms

## Preliminary Checks

Checks performed:
- Exact-name web searches for obvious existing apps or brands
- RDAP domain checks for `.com`, `.app`, and `.ai`

Important:
- This is not legal trademark clearance.
- Domain “not found” means the RDAP registry did not show an active registration during this check.
- A registrar can still mark a domain as premium, reserved, or unavailable at checkout.
- Before launch, run formal trademark and social handle checks.

## Recommended Shortlist

### 1. CupSkies

Why it works:
- Best fit for the World Cup launch.
- Owns the playful forecast/weather vibe.
- Short and easy to remember.
- Does not sound like betting.
- Works nicely in English, Spanish, and French.

Preliminary domain check:
- `cupskies.com`: not found
- `cupskies.app`: not found
- `cupskies.ai`: not found

Recommendation:
- Best current working name.

## Top Recommendation

Use **CupSkies** for the World Cup MVP.

Reason:
- It is fun, ownable, clear enough, and matches the weather-forecast tone.
- The `.com`, `.app`, and `.ai` domains looked open in the preliminary RDAP check.
- It gives us a strong event-focused launch brand.

If we want a name that can stretch more naturally into NFL Fantasy, use **MatchSkies**.

## Next Checks Before Committing

Before buying domains or designing the logo:
- Check domain purchase availability at a registrar.
- Check major social handles.
- Check USPTO trademarks.
- Check app stores.
- Check common misspellings.
- Confirm pronunciation in English, Spanish, and French.
